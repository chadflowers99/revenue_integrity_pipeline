# Demo Retail Case

This folder contains a demo client setup for the forensic revenue pipeline.

## Files

- `client_config.py`: Client-specific cleanup and projection configuration.
- `raw_data.csv`: Sample retail sales data with both clean and intentionally messy rows.
- `output/`: Destination for cleaned outputs and projection reports.

## Usage

1. Point your pipeline runner to `demo_retail_case`.
2. Run the pipeline.
3. Review generated files in `output/`.
